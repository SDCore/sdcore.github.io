Current Project
myHub
myHub is an in-progress project that focuses on giving specific information in one place for the user to use. At the moment, not muc is there. Weather is pulled from the DarkSky API, and directions are using Google Maps Directions API and Google Maps Geocoding API. In the future, I hope to have more information, such as RSS Feeds, Calender Events from a users Google Account, and more!
